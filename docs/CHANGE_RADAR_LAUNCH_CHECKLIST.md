# Change Radar - Pre-Launch Checklist

## Product Readiness

### Project Intelligence Core
- [ ] Multi-project portfolio dashboard - tested with 20+ projects
- [ ] Project health scoring displays correctly (AI-generated)
- [ ] Risk register with severity/likelihood matrix working
- [ ] Deliverables tracker shows dependencies correctly
- [ ] Resource allocation view accurate
- [ ] Timeline/milestone tracking functional

### AI Insights Engine
- [ ] Automated risk detection triggering correctly
- [ ] "Projects at risk" predictions showing with clear reasoning
- [ ] Resource conflict warnings appearing when expected
- [ ] Trend analysis charts rendering properly
- [ ] Natural language query returning accurate results
- [ ] AI response times acceptable (<3 seconds)

### Automated Reporting
- [ ] One-click executive summary generates correctly
- [ ] PDF export looks professional/board-ready
- [ ] Slide export formatting correct
- [ ] Report templates customizable
- [ ] Scheduled reports sending reliably
- [ ] Report history accessible

### Data Input
- [ ] Manual entry UI is fast and intuitive
- [ ] Bulk CSV/Excel import works without errors
- [ ] Import handles edge cases (empty fields, special characters)
- [ ] Data validation prevents bad inputs

### Collaboration
- [ ] Role-based access working (Executive vs PM views)
- [ ] Comments/notes saving and displaying
- [ ] Activity log capturing all changes
- [ ] Notifications triggering correctly
