## The Problem

Programme offices drown in data but starve for insight. Teams spend weeks preparing SteerCo packs, manually correlating project health across dozens of workstreams. By the time issues surface in meetings, they've already metastasized.

I built Change Radar to flip this dynamic: **predict problems before they become crises**.

## The Build

Change Radar pulls project data from multiple sources into a unified predictive view. The AI analyzes patterns across timeline slips, resource allocation, stakeholder sentiment, and dependency health to surface risks before they impact delivery.

### Core Features

**Smart Data Ingestion**

- Connects to Jira, Asana, Monday, and custom APIs
- Real-time sync with automatic conflict resolution
- Smart mapping between different project schemas

**Predictive Analytics Engine**

- Risk scoring based on 15+ leading indicators
- Dependency chain analysis for cascade impact
- Natural language risk explanations

**Automated Reporting**

- One-click SteerCo pack generation
- Custom dashboards for different stakeholder needs
- Slack/Teams integration for proactive alerts

## Technical Deep Dive

The architecture prioritizes speed and reliability:

```typescript
// Risk prediction pipeline
const analyzeProjectHealth = async (projectId: string) => {
  const [timeline, resources, dependencies] = await Promise.all([
    getTimelineData(projectId),
    getResourceData(projectId),
    getDependencyGraph(projectId),
  ]);

  const riskScore = await predictRisk({
    timeline,
    resources,
    dependencies,
    historicalPatterns: await getPatterns(projectId),
  });

  return {
    score: riskScore,
    factors: getRiskFactors(riskScore),
    recommendations: await generateRecommendations(riskScore),
  };
};
```
