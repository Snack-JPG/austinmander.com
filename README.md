# austinmander.com

Personal brand website for Austin Mander - AI Systems Builder.

**Positioning:** "I build AI systems that actually work."

---

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Site Structure

```
/                  Homepage (hero + proof + featured project + CTA)
/work              Projects and case studies
/about             Personal story and beliefs
/services          Service offerings
/contact           Calendly + email
```

---

## Design Direction

- **Aesthetic:** Builder, not marketing
- **Mode:** Dark theme
- **Typography:** Monospace headings, clean sans-serif body
- **Content:** Proof-first, outcome-focused
- **Speed:** Fast, minimal, mobile-first
